# Stream H：周縁的・運動的組織形態の系譜

## 概要

主流類型論（Weber、Mintzberg、法人格分類）から漏れる組織群——協同組合、NPO・社会的企業、女性組織、社会運動、インフォーマル経済、先住民組織、デジタルコモンズ——を系譜的に整理する。これら周縁的組織は「成員=所有者」「ミッション駆動」「自治・横断・水平」「集合的アイデンティティ」「非分配制約」など、株式会社/官僚制では捕捉不能な構造原理を体現し、組織を生命として捉える視座へ豊かな材料を提供する。

## 各カテゴリの主要事例

### 1. 協同組合・相互扶助の系譜

**1) Rochdale Equitable Pioneers Society（1844–現存）**
- 形態：消費者協同組合の祖型（28人の織工、英Rochdale）
- 規模：開業時資本£28、現代協同組合運動約30億人に展開（ICA系）
- コア機能：低価格食料品の組合員向け小売、剰余の取引比例配当
- 持続性の源泉：Rochdale Principles（一人一票、開放加入、利息制限、教育、政治宗教中立、現金取引、剰余配当）の普遍性
- 既存類型論との関係：株式会社（資本×投票）でも官僚制でもない「成員=所有者=利用者」の三位一体。Mintzbergの五構造に該当しない
- 典拠：[ICA - The Rochdale Pioneers](https://ica.coop/en/rochdale-pioneers); [Wikipedia - Rochdale Society](https://en.wikipedia.org/wiki/Rochdale_Society_of_Equitable_Pioneers)

**2) Raiffeisen信用組合（1864–現存）**
- 形態：農村相互信用組合（独Neuwied）
- 規模：1869年までに独で約100組合、現代は世界各国の信用組合の祖型
- コア機能：高利貸搾取からの農民解放、組合員預金を組合員融資に
- 持続性の源泉：「Selbsthilfe, Selbstverwaltung, Selbstverantwortung」（自助・自治・自責）の三S、無限連帯責任
- 既存類型論との関係：資本市場を介さない貯蓄-融資マッチング、地域限定原則（locality principle）
- 典拠：[ICA - Raiffeisen](https://ica.coop/en/friedrich-wilhelm-raiffeisen); [Wikipedia](https://en.wikipedia.org/wiki/Friedrich_Wilhelm_Raiffeisen)

**3) Mondragón Corporation（1956–現存）**
- 形態：労働者協同組合連合（バスク）
- 規模：約8万人雇用、約95社、Caja Laboral（1959）、Lagun Aro（1966）含む
- コア機能：労働主権、Don José María Arizmendiarrietaの司牧的教育起源
- 持続性の源泉：10原則（労働の主権、資本の従属、給与連帯、相互協同、社会変革、普遍性、教育）、利益の内部再投資・連帯基金
- 既存類型論との関係：労働者所有 + 多stakeholder + 教育機関（Mondragón Univ.）の統合エコシステム
- 典拠：[Mondragon Corporation](https://www.mondragon-corporation.com/en/history/); [Wikipedia](https://en.wikipedia.org/wiki/Mondragon_Corporation)

**4) International Cooperative Alliance（ICA, 1895–現存）**
- 形態：協同組合の国際連合体（本部ブリュッセル）
- 規模：306連合体、105ヶ国、約10億人、世界に約300万協同組合（2024）
- コア機能：協同組合原則の維持・国際代表
- 持続性の源泉：価値・原則の継承、4地域局・8セクター組織・5委員会の入れ子構造
- 典拠：[ICA About](https://ica.coop/en/about-us/international-cooperative-alliance); [ICA Annual Report 2024](https://ica.coop/en/media/library/annual-reports-report-activities/ica-annual-report-2024)

### 2. NPO・社会的企業の系譜

**5) Hull-House / Settlement House Movement（1889–）**
- 形態：定住型社会改革コミュニティ（米Chicago、Jane Addams & Ellen Starr）
- 規模：1900年に米国で100館、1920年に約500館
- コア機能：移民労働者地区での教育・芸術・調査・政策提言
- 持続性の源泉：「住み込み」原理（resident）、宗教非依存・民主的理念、女性大学卒業生のキャリア基盤
- 既存類型論との関係：プロボノ住居+調査機関+市民形成の混合、近代福祉国家・社会調査の母胎
- 典拠：[Hull-House Museum](https://www.hullhousemuseum.org/about-jane-addams); [Wikipedia - Settlement movement](https://en.wikipedia.org/wiki/Settlement_movement)

**6) 米国 Tax Reform Act 1969（NPO法体系）**
- 形態：法的二分体系（public charity / private foundation）
- 規模：米国財団部門全体に課税ルール導入
- コア機能：自己取引禁止、5%最低支出（distribution）、政治活動禁止、開示義務
- 持続性の源泉：免税特権と公開性の交換による正当性確保
- 既存類型論との関係：NPOを「非分配制約 + ミッション縛り」で再定義、Hansmannの非分配制約論の制度実装
- 典拠：[Wikipedia - TRA 1969](https://en.wikipedia.org/wiki/Tax_Reform_Act_of_1969); [HistPhil](https://histphil.org/2020/01/22/the-private-foundation-rules-at-fifty-how-did-we-get-there/)

**7) B Corporation（B Lab認証, 2006–現存）**
- 形態：第三者認証付きハイブリッド企業
- 規模：9,576社、102ヶ国、160業界（2025年3月）
- コア機能：B Impact Assessmentで80点以上、定款にステークホルダー条項を組込
- 持続性の源泉：法人格（営利）と認証（社会）の二層構造、benefit corporation法（米州法）との連動
- 既存類型論との関係：株式会社×ミッション縛りのハイブリッド、Friedmanのshareholder primacyへの代替
- 典拠：[Wikipedia - B Corporation](https://en.wikipedia.org/wiki/B_Corporation_(certification)); [B Lab](https://www.bcorporation.net/en-us/certification/)

**8) Social Impact Bond（Peterborough, 2010–）**
- 形態：成果連動型官民資金調達ストラクチャ
- 規模：£500万、17投資家（trusts, foundations）、3,000受刑者対象
- コア機能：再犯減少→政府支払、Social Finance Ltd.が組成
- 持続性の源泉：成果検証（control group比較）、9%再犯減で年率約3%リターン
- 既存類型論との関係：契約・組織・債券の境界線上、組織形態というより「ガバナンス・ストラクチャ」
- 典拠：[Wikipedia - SIB](https://en.wikipedia.org/wiki/Social_impact_bond); [Social Finance UK](https://www.socialfinance.org.uk/people/what-we-do/reducing-reoffending-in-peterborough)

### 3. 女性組織と女性運営組織

**9) Beguinage（13世紀–18世紀、現存物理遺構）**
- 形態：誓願なき女性宗教共同体（北西欧、特に低地諸国）
- 規模：Ghent数千人、Paris最大400人、独で60–70人規模の家
- コア機能：祈り・労働（紡織業）・福祉、未婚・寡婦の自立空間
- 持続性の源泉：母会なし、各 beguinage 独立、Grande Dame民主選出、財産個人保有可
- 既存類型論との関係：修道院（誓願・閉鎖）と世俗（家族・市場）の中間、ジェンダー史で「修道院に行きたくない女性のための制度」
- 典拠：[Wikipedia - Beguines](https://en.wikipedia.org/wiki/Beguines_and_Beghards); [Britannica - Beguines](https://www.britannica.com/topic/Beguines)

**10) WCTU - Woman's Christian Temperance Union（1874–現存）**
- 形態：女性専有の全国社会改革団体（米Cleveland発）
- 規模：初期5年で1,000以上の地方支部、19世紀末に世界最大の女性組織
- コア機能：禁酒・教育・参政権・労働改革（"Do Everything"政策）
- 持続性の源泉：女性専有の演壇・役職・指導訓練、宗教ネットワーク
- 既存類型論との関係：単一目的団体ではなく「保護→拡張」型のagenda広がり、女性公的役職訓練機関
- 典拠：[Wikipedia - WCTU](https://en.wikipedia.org/wiki/Woman's_Christian_Temperance_Union); [Britannica](https://www.britannica.com/topic/Womans-Christian-Temperance-Union)

**11) SEWA - Self-Employed Women's Association（1972–現存）**
- 形態：インフォーマル経済女性労働組合（印Ahmedabad、Ela Bhatt）
- 規模：378万人、20州、71協同組合、SEWA Bank（1974, microfinance祖型）
- コア機能：露天商・家内労働者・建設労働者の組織化、銀行・保険・住宅の自前供給
- 持続性の源泉：Gandhi系TLA女性部門起源、組合 + 協同組合 + 開発NGOのハイブリッド
- 既存類型論との関係：労組（formal）でも協同組合でもNGOでもない「インフォーマル労働の重ね組織化」
- 典拠：[SEWA](https://www.sewa.org/about-us/history/); [Wikipedia - SEWA](https://en.wikipedia.org/wiki/Self_Employed_Women's_Association)

**12) Gender Lens Investing（2009–現存、用語成立）**
- 形態：投資戦略・ファンド運用の手法
- 規模：私募約79億ドル、公募ジェンダーレンズ株式44本、約42.7億ドル（2023）
- コア機能：女性所有/職場ジェンダー公平/女性向け製品への資本配分
- 持続性の源泉：Criterion Institute、Suzanne Biegel、GIIN等のフィールドビルダー、SDG5連動
- 既存類型論との関係：単一組織ではなく「投資論理（lens）」だが、ファンド構造（GP/LP）に組織形態として埋め込まれる
- 典拠：[Wikipedia - Gender Lens Investing](https://en.wikipedia.org/wiki/Gender_lens_investing); [Cambridge Associates](https://www.cambridgeassociates.com/wp-content/uploads/2018/09/Gender-Lens-Investing.pdf)

### 4. 社会運動・抗議・反システム組織

**13) Charles Tillyの社会運動レパートリー論（理論・1768–現代）**
- 形態：分析枠組（社会運動を「キャンペーン+レパートリー+WUNC」と定義）
- コア機能：worthiness, unity, numbers, commitmentの公的提示
- 既存類型論との関係：組織の境界より「持続的公的請求行動」を単位とし、組織の中心性を相対化
- 典拠：[Wikipedia - Charles Tilly](https://en.wikipedia.org/wiki/Charles_Tilly); [Social Movements 1768-2004 (PDF)](https://voidnetwork.gr/wp-content/uploads/2016/09/Social-Movements-1768-2004-by-Charles-Tilly.pdf)

**14) SDS - Students for a Democratic Society（1962–1969）**
- 形態：参加型民主主義を綱領とする学生運動組織（Port Huron Statement, 1962）
- コア機能：大学・公民権・反戦の連携、Tom Hayden中心起草
- 持続性の源泉：「個人が自己決定に関与する」原理、地方支部の自律
- 既存類型論との関係：会員組織+運動+知的サークルの混合、1969分裂で消失するが「参加型民主主義」概念が継承
- 典拠：[Wikipedia - Port Huron Statement](https://en.wikipedia.org/wiki/Port_Huron_Statement); [Teaching American History](https://teachingamericanhistory.org/document/port-huron-statement/)

**15) SCLC vs SNCC（1957/1960–）**
- SCLC：教会指導者中央集権、King象徴、全国規模キャンペーン（Birmingham, Selma）
- SNCC：Ella Baker触発の「集団中心」非父権モデル、地方コミュニティ組織化、コンセンサス志向
- 持続性の源泉の対比：カリスマ・メディア vs 地域埋め込み・草の根
- 既存類型論との関係：同じ運動内に「中央集権ミッション型」「分散参加型」が並存→運動内多形態性の原典例
- 典拠：[King Institute - SNCC](https://kinginstitute.stanford.edu/student-nonviolent-coordinating-committee-sncc); [Wikipedia - SCLC](https://en.wikipedia.org/wiki/Southern_Christian_Leadership_Conference)

**16) Occupy Wall Street（2011）**
- 形態：水平・無リーダー・修正コンセンサス型General Assembly
- 規模：NYC Zuccotti Park発、世界80か国以上に拡散、70+ working groups
- コア機能：意思決定はGA + spokes council、progressive stack、9/10投票フォールバック
- 持続性の源泉：アナーキスト系の組織技法、SNS拡散
- 既存類型論との関係：法人格なし、雇用なし、所有なしの「過渡的集合体」、形態論的にはほぼ純粋な集合知実験
- 典拠：[Wikipedia - OWS](https://en.wikipedia.org/wiki/Occupy_Wall_Street); [General Assembly](https://en.wikipedia.org/wiki/General_assembly_(Occupy_movement))

**17) Black Lives Matter（2013–）**
- 形態：ハッシュタグ起源の分散ネットワーク + BLM Global Network Foundation（法人）
- 規模：40+チャプター、3人理事会、ハッシュタグ#BlackLivesMatter（Cullors, 2013）
- コア機能：地方自律 + 共通原則、デジタル協調
- 持続性の源泉：象徴的ハッシュタグ、運動と法人の二層構造
- 既存類型論との関係：「運動」と「組織」の混同を制度的に問題化、財務透明性論争、デジタル時代の運動形態の典型
- 典拠：[Wikipedia - BLM](https://en.wikipedia.org/wiki/Black_Lives_Matter); [Wikipedia - BLMGNF](https://en.wikipedia.org/wiki/Black_Lives_Matter_Global_Network_Foundation)

**18) Extinction Rebellion（2018–）**
- 形態：自己組織化システム（Holacracy触発）の役割×サークル構造
- コア機能：affinity group・working group、命令系統なし、合意は「同意ベース」
- 持続性の源泉：自治の明文化された手続（Self-Organising System文書）
- 既存類型論との関係：Holacracyを社会運動規模に転用した稀有例、企業由来ガバナンス技法の運動への移植
- 典拠：[Wikipedia - XR](https://en.wikipedia.org/wiki/Extinction_Rebellion); [Quartz - XR Holacracy](https://qz.com/work/1776861/extinction-rebellion-is-using-holacracy-to-scale-its-international-movement)

### 5. インフォーマル経済組織

**19) Hernando de Soto / 「死せる資本」論（1986/2000）**
- 形態：理論枠組 + 政策実装（ILD, Lima）
- コア機能：インフォーマル財産の法制度化＝「死せる資本」を信用化
- 既存類型論との関係：組織論というより「組織の前提条件としての所有制度」、批判的にはインフォーマル組織化の集合的実践を看過
- 典拠：[Wikipedia - de Soto](https://en.wikipedia.org/wiki/Hernando_de_Soto_(economist)); [Lincoln Institute](https://www.lincolninst.edu/publications/articles/influence-sotos-mystery-capital/)

**20) StreetNet International（2002–現存）**
- 形態：路上商人グローバル連盟（南ア Berea本拠）
- 規模：881,400人、61加盟、55ヶ国（2024）
- コア機能：労働者統制ボトムアップ構造、SEWAに触発
- 持続性の源泉：union/cooperative/associationの異種を許容する加盟構造
- 既存類型論との関係：union（雇用関係前提）でない労働者組織、ICA系協同組合とも別の「インフォーマル労働者」固有形態
- 典拠：[StreetNet International](https://streetnet.org.za/who-we-are/); [ILO](https://www.ilo.org/resource/article/streetnet-international-organizing-street-vendors-towards-formalization-and)

### 6. 先住民・マイノリティ組織

**21) Haudenosaunee Confederacy（伝統的議会、1142年伝承–現存）**
- 形態：6国連合（Mohawk, Oneida, Onondaga, Cayuga, Seneca, Tuscarora）
- 規模：56 Chiefs（hoyaneh）+ 49 Clan Mothers（iakoia:ne）
- コア機能：Great Law of Peace（Gayanesshagowa）下のコンセンサス意思決定
- 持続性の源泉：Clan Mother（女性）が Chief（男性）を選任・解任、世襲と功績のハイブリッド
- 既存類型論との関係：国家でも企業でもないfederated council、米国憲法起草に影響したとされる（議論あり）
- 典拠：[Haudenosaunee Confederacy](https://www.haudenosauneeconfederacy.com/government/); [Wikipedia - HCCC](https://en.wikipedia.org/wiki/Haudenosaunee_Confederacy_Chiefs_Council)

**22) Alaska Native Corporation（ANCSA, 1971）**
- 形態：先住民権利の和解として議会創設の私営営利株式会社（12地域 + 200村）
- 規模：約2,700万エーカー、約14万株主
- コア機能：先住民権利を株式所有に変換、地域corp（地下権）vs 村corp（地上権）
- 持続性の源泉：法定の特殊corp、株式譲渡制限、配当
- 既存類型論との関係：「先住民権 = 営利法人株式」という前例なき変換、tribal governmentと別建ての民事法人
- 典拠：[Wikipedia - ANC](https://en.wikipedia.org/wiki/Alaska_Native_corporation); [ANCSA Regional Association](https://ancsaregional.com/about-ancsa/)

**23) GIDA - Global Indigenous Data Alliance / CARE Principles（2019–）**
- 形態：先住民データ主権の国際ネットワーク
- コア機能：CARE = Collective Benefit, Authority to Control, Responsibility, Ethics
- 持続性の源泉：FAIR原則（findable等）の権力非対称性批判、Stephanie Russo Carroll & Maui Hudson主導
- 既存類型論との関係：データ governance に先住民集合権を埋め込む、ISO/IEC的標準化への対抗軸
- 典拠：[GIDA](https://www.gida-global.org); [Nature Scientific Data - CARE/FAIR](https://www.nature.com/articles/s41597-021-00892-0)

### 7. デジタル運動とコモンズ

**24) Wikipedia / Wikimedia Foundation（2001/2003–）**
- 形態：自治的編集者コミュニティ + 支援財団（501(c)(3)）
- コア機能：editor / admin / bureaucrat / Arbitration Committeeの段階的責任、財団は技術・組織インフラのみ
- 持続性の源泉：政策の community consensus 形成、財団がコンテンツに介入しない自己拘束
- 既存類型論との関係：財団（法人）と community（非法人）の二層、編集者は雇用関係なし
- 典拠：[Wikimedia Foundation Governance](https://foundation.wikimedia.org/wiki/Home); [Wikipedia - Formal organization](https://en.wikipedia.org/wiki/Wikipedia:Formal_organization)

**25) Apache Software Foundation（1999–）**
- 形態：meritocracyベース 501(c)(3) 公益法人
- コア機能：user→developer→committer→PMC member→PMC chair→ASF memberの実績昇格
- 持続性の源泉：個人ベース投票（雇用主・寄付額無関係）、PMC毎の自治、Boardは組織監督のみ・技術非介入
- 既存類型論との関係：「コード貢献」という非経済的指標で投票権配分、寄付者影響を意図的に遮断
- 典拠：[Apache - How it works](https://www.apache.org/foundation/how-it-works/); [Wikipedia - ASF](https://en.wikipedia.org/wiki/The_Apache_Software_Foundation)

**26) Linux Kernel（1991–）+ Linux Foundation（2007–）**
- 形態：BDFL（Torvalds）+ maintainer階層 + Linux Foundation（インフラ・人件費）
- コア機能：パッチがmaintainer階層を上昇、最終マージはTorvalds、技術紛争の最高調停
- 持続性の源泉：技術判断と組織支援の分離、2025末「Linux Kernel Continuity Document」で承継計画
- 既存類型論との関係：BDFL型は単一個人依存だが、財団との分離で institutional resilience を確保
- 典拠：[Wikipedia - Linux Foundation](https://en.wikipedia.org/wiki/Linux_Foundation); [BigGo News - Succession](https://biggo.com/news/202601282221_linux-kernel-succession-plan-linus-torvalds)

**27) Ostrom 8原則とコモンズガバナンス（理論・1990–）**
- 形態：Common-Pool Resourceの設計原理（Governing the Commons, 1990）
- 8原則：境界明確化／文脈適合ルール／参加型決定／監視／段階的制裁／紛争解決アクセス／組織化権承認／入れ子事業
- 既存類型論との関係：市場でも国家でもない「第三の道」、私有/公有の二項を超える組織原理
- 典拠：[SESMAD - Ostrom](https://sesmad.dartmouth.edu/theories/54); [Governing the Commons (PDF)](https://www.actu-environnement.com/media/pdf/ostrom_1990.pdf)

**28) Creative Commons（2001–）**
- 形態：法的ツール提供 501(c)(3) + Global Network（Chapter化、2018再構成）
- コア機能：CC BY/SA/NC/NDライセンスの法律技術提供、各国法域で翻訳・適応
- 持続性の源泉：Lessig創設の理念、Stanford Law起源、ライセンスを法律的「公共財」化
- 既存類型論との関係：「組織」より「法的インフラ + 国際ネットワーク」、法律家・著者・利用者の三方インターフェース
- 典拠：[Wikipedia - Creative Commons](https://en.wikipedia.org/wiki/Creative_Commons); [CC Governance](https://creativecommons.org/governance/)

## 主流類型論で捕捉できない理由

1. **法人格と組織実体の乖離**：BLM、Occupy、XRなどは「組織」と呼べる法人と「運動」としての行為主体性が分離。Mintzbergの五構造（simple/machine/professional/divisional/adhocracy）が前提とする「単一実体」が成り立たない。

2. **所有・利用・労働の合一**：協同組合は株主-顧客-労働者の境界を融合。法人格分類（営利/非営利、株式会社/一般社団）は所有原理（一人一票 vs 一株一票）を主軸に立てない。

3. **メリット/長老/性別による権威割当**：Apache のmeritocracy、Haudenosaunee のClan Mother制、Beguinage のGrande Dame は、株主投票でも国家委任でもない権威源。

4. **入れ子・連邦・ネットワーク**：ICA、StreetNet、SEWAの「組合の組合」、Mondragónの「協同組合グループ」、Ostromの「nested enterprises」など、単一階層ピラミッドではない。

5. **デジタル化された集合的アイデンティティ**：ハッシュタグ運動、コミット履歴に基づく権威、ライセンスによる遠隔結合は、地理・対面・法的契約を前提とする伝統的組織論で扱えない。

6. **インフォーマル/フォーマル境界横断**：SEWA、StreetNetは「正規化」を目的とせず、インフォーマル領域での集合行為を組織化する。de Soto的フォーマライゼーション論はこの集合実践を看過する。

7. **時間スケールの多様性**：Beguinageの数百年、Occupyの数ヶ月、SIBの単発契約、CCライセンスの永続性——「組織の寿命」を単一尺度で扱えない。

## ボトムアップ分析・データベース設計への含意

周縁から見出される追加次元：

- **所有原理（ownership principle）**：株式比例 / 一人一票 / 利用比例 / 集合的（先住民・コミュニティ）/ 不在（コモンズ）/ 信託
- **権威源泉（authority source）**：資本 / 選挙 / 任命 / メリット（コード/実績）/ 世襲 + 解任権 / カリスマ / 合意
- **境界透過性（boundary permeability）**：契約雇用 / 会員 / volunteer / contributor / hashtag participant / shareholder
- **意思決定モード**：majority vote / consensus / modified consensus / BDFL / lazy consensus / spoke council
- **持続性メカニズム**：法人格 / 物理空間（beguinage の壁、修道院）/ 儀式・原則文書 / コードリポジトリ / ハッシュタグ / 法的ライセンス
- **正当性源泉**：法律 / ミッション / 受益者承認 / メリット / 伝統 / 集合的アイデンティティ / 成果（SIB）
- **ハイブリッド軸**：営利-非営利、運動-法人、地域-グローバル、対面-デジタル を連続体として
- **「組織」と「インフラ」の区別**：CCのライセンス、Ostrom原則、SIB契約は「組織」というより「組織化を可能にするインフラ」
- **女性・先住民・労働者の集合的主体性**：個人会員制を超える「集合的成員性」の概念設計

## 倫理・センシティブな配慮

1. **Indigenous Data Sovereignty**：CARE原則に従い、先住民組織のデータは Collective Benefit, Authority to Control, Responsibility, Ethics を尊重。FAIR原則だけで開示・標準化を進めない。Haudenosaunee、ANCSA等は外部研究者の枠組ではなく当事者の語りを優先。

2. **運動の安全性**：BLM、SNCC、Occupy等の活動家・組織者の名前・詳細を一次史料以上に記述する際、現存活動家の安全（監視・嫌がらせ）に配慮。匿名化されている場合は匿名のまま扱う。

3. **女性組織の歴史的不可視化への対抗**：WCTU、SEWA、Beguinage、Hull-Houseは男性中心の組織論で過小評価されてきた。「補助的」「準組織的」と扱わず、独立した形態として位置づける。

4. **インフォーマル経済の主体性尊重**：de Soto的「フォーマル化すべき対象」視座と、SEWA・StreetNet的「集合的主体」視座の緊張を明示。組織化を「正規化への過程」として規範化しない。

5. **西洋近代中心主義の相対化**：「組織」概念自体が法人格・契約を前提とした西洋近代の産物であり、Haudenosaunee Grand Council のような数百年継続する非西洋形態を「前近代」「非組織」とラベリングしない。

6. **運動の commodification 警戒**：B Corp、SIB、Gender Lens Investing は社会的ミッションの金融化形態。市場化と価値の希薄化（mission drift）の危険性を含意として記述。
